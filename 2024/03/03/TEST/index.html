(1) 풀이 2종오류의 확률을 구하라 유의수준 5%

2종오류확률은 $P(H_0 채택|H_1 이 맞는 상황)$ 이다,

상황을 점검하면 아래와 같다.

위 문제에서 귀무가설과 대립가설은 $H_0 = 1500$ , $H_1=1600$ 이다

기각역은 검정통계량이 $z_{\alpha_{0.05}}$ 보다 작은거

$H_0$ 검정 통계량을 $T$ 라고하면 $T = \frac{\bar{x}-1500}{\frac{200}{\sqrt{n}}}$ 이고

정리하면 $H_0$ 기각역은 ${ T> z_{\alpha_{0.05}}}$ 이다.

이제 문제를 풀면

$$
\begin{aligned}
P(H_0 채택|H_1 이 맞는 상황) &= P(H_0\ 기각역에 \ 안속함 , \mu = 1600) \
& = P( T \leq z_{\alpha_{0.05}} ,\mu = 1600 ) \
& = P( \frac{\bar{x}-1500}{\frac{200}{\sqrt{n}}} \leq z_{\alpha_{0.05}} ,\mu = 1600 ) \
그러나 H_1 상에서 진짜 평균은 1600 이고 &이상황이면 -1500 을하면 정규분포가 되는게아니라 -1600을 해야 정규 분포를 따른다.\
&= P( \frac{\bar{x}-1600+100}{\frac{200}{\sqrt{n}}} \leq z_{\alpha_{0.05}} ,\mu = 1600 ) \
&= P( Z+\frac{100}{\frac{200}{\sqrt{30}}} \leq z_{\alpha_{0.05}} ) \
&= P( Z+\frac{\sqrt{30}}{2} \leq z_{\alpha_{0.05}} ) \

&= P( Z \leq z_{\alpha_{0.05}} - \frac{\sqrt{30}}{2} ) \
&= P( Z \leq 1.645- 2.738613 ) \
\end{aligned}
$$
